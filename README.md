**Column Chromatography Simulator (カラム精製・分子シミュレーター)**

イオン交換クロマトグラフィーの仕組みを、ブラウザ上で直感的に学習できるインタラクティブな物理シミュレーターです。

🚀 **概要**

このプロジェクトは、生化学実験で用いられる「イオン交換クロマトグラフィー」におけるタンパク質分離の原理を可視化したものです。
環境のpHや塩濃度（NaCl）を操作することで、各分子がどのように樹脂に吸着し、あるいは溶出していくかをリアルタイムのアニメーションで観察できます。

✨ **特徴**
- **直感的な操作**: pH、NaCl濃度、流速などをスライダーで動かすと、即座に分子の挙動に反映されます。
- **動的な電荷計算**: タンパク質ごとの等電点（pI）と溶液のpHの差に基づき、各分子の電荷（＋ / －）をリアルタイムに算出します。
- **物理挙動の再現**: HTML5 Canvasを用いて、分子の拡散（ゆらぎ）や樹脂への吸着エフェクトを滑らかに表現しています。
- **導入が簡単**: HTML/CSS/JavaScriptの1ファイルで完結しているため、ダウンロードしてブラウザで開くだけで動作します(オフライン使用できます)。

🧪 **学習できる科学的原理**
このシミュレーターを通じて、以下の概念を視覚的に学ぶことができます。
1. **等電点（pI）と電荷の関係**
    - pH < pI のとき：タンパク質は正（＋）に帯電
    - pH > pI のとき：タンパク質は負（－）に帯電
2. **イオン交換の種類**
    - 陽イオン交換: 負に帯電した樹脂が、正に帯電した分子を捉えるプロセス。
    - 陰イオン交換: 正に帯電した樹脂が、負に帯電した分子を捉えるプロセス。
3. **塩（NaCl）による溶出の仕組み**
    - 高濃度の塩イオンが樹脂の結合部位を競合し、吸着していた分子を脱離させるプロセス。
🛠 **使い方**
1. index.html をブラウザ（Chrome, Edge, Firefox, Safari等）で開きます。
2. **1.カラム設定**: 陽イオン交換か陰イオン交換かを選択し、サンプル数を決定します。
3. **2.移動相・溶出条件**: バッファーのpHや流速を設定します。
4. **3.タンパク質設定**: 各サンプルの等電点（pI）を調整します。
5. **実験開始**: 「コックを開ける」をクリックするとサンプルが流下します。
6. **分離・溶出**: pHを変化させたり、「NaCl 溶出」を行ったりして、目的の分子を分離するシミュレーションを行います。

🔧 **カスタマイズ（開発者向け）**
ソースコード内の以下の数値を変更することで、挙動を調整可能です。
  - ```PARTICLES_PER_GROUP```: 1グループあたりの粒子の数
  - ```BASE_SPEED```: 分子の流下速度
  - ```draw()``` 関数内の条件式：吸着や離脱の感度を調整できます
